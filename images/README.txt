SUMMIT COATINGS LLC — IMAGE ASSET GUIDE
========================================

Drop your real photos into this /images/ folder using the exact filenames listed
below. The website's HTML already points to these filenames — no code edits
needed. Until a real file is present, the site shows a clearly-labeled
"PLACEHOLDER" tile so you can see exactly what goes where.

------------------------------------------

------------------------------------------
  logo.svg    — Site logo (currently a placeholder SVG; replace with your real

                <img src="images/logo.svg"> tags in the HTML to logo.png).

GENERAL TIPS
------------------------------------------
- JPG quality 75-85 is plenty for the web.
- Resize photos to ~1600px on the long edge before uploading (a 6 MB photo
  from a phone is way more than the site needs).
- Free tools: Squoosh.app (Google) for compression; Photopea.com for resizing.
- Take "before" photos at every job from now on — they make the strongest
  portfolio content alongside the finished shots.
